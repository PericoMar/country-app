Buscar país por ID

Recuerda que puedes buscar un país utilizando su ID en la URL. Utiliza la ruta /countries/:id y reemplaza :id con el ID del país que deseas buscar.

Ejemplo:

Para buscar España, puedes utilizar el ID del país o el código ISO Alpha-2. Por ejemplo, para España, el código ISO Alpha-2 es ES.

Para buscar por ID, la URL sería: /countries/ES.

Buscar país:

  • España (ES)
  • Estados Unidos (US)
  • Francia (FR)
  • Alemania (DE)
  • Italia (IT)

Tecnologías

Angular 17 ha sido el principal framework utilizado en el desarrollo de la parte front de mi aplicación. Este potente framework de desarrollo frontend ha proporcionado una estructura sólida y modular para la construcción de una experiencia de usuario dinámica y receptiva.

Uno de los aspectos más destacados del uso de Angular ha sido su capacidad para gestionar las solicitudes y respuestas de la API de manera eficiente. Los servicios de Angular han sido fundamentales para establecer la comunicación con la API REST Countries, permitiendo que mi aplicación obtenga y muestre información detallada de países de todo el mundo en tiempo real.

Además, Angular's Router Module ha sido esencial para la gestión de las rutas de mi aplicación. Con él, he podido definir y organizar las diferentes vistas y componentes de mi aplicación, facilitando la navegación del usuario a través de las diversas secciones y funcionalidades.

En resumen, Angular ha sido una pieza fundamental en el desarrollo de mi aplicación, proporcionando las herramientas necesarias para crear una experiencia de usuario fluida y atractiva, así como para establecer una sólida conexión con la API que impulsa mi aplicación de búsqueda de países.



Bootstrap 5 ha sido una herramienta indispensable en el desarrollo de mi aplicación. La versatilidad de sus componentes y la conveniencia de sus clases predefinidas han simplificado enormemente el proceso de diseño y desarrollo de la interfaz de usuario. Desde la creación de diseños receptivos hasta la estilización de elementos de navegación y formularios, Bootstrap ha proporcionado una base sólida sobre la cual construir mi aplicación de búsqueda de países.

La utilización de los componentes de Bootstrap, como las barras de navegación, las tarjetas y los dropdowns, ha mejorado significativamente la experiencia del usuario al proporcionar elementos visuales coherentes y atractivos. Además, la integración de las clases de Bootstrap en el diseño y el estilo de los elementos ha agilizado el proceso de desarrollo, permitiéndome centrarme en la funcionalidad de la aplicación sin preocuparme por los detalles de diseño.

En resumen, Bootstrap ha sido una herramienta invaluable en mi caja de herramientas de desarrollo web, permitiéndome crear una interfaz de usuario pulida y profesional de manera eficiente y efectiva. Su amplia gama de componentes y clases ha hecho posible llevar a cabo una variedad de tareas de diseño y estilización con facilidad, lo que ha contribuido en gran medida al éxito de mi proyecto de búsqueda de países.
